接口文档

用户模块​

登录​

简要描述​
  • 登录
请求URL​
  • https://xxxxxxxxx/qiniuyun/user/login
请求方式​
  • Post
请求body参数​
参数名必选类型说明
email是string用户名
password是string密码
返回示例--成功​
  {
    "status": 10000,
    "info": "success"，
    "data":{
        "Token":"xxxx",
        "name":"用户名",
        "useId":001,
        "email":'xxx.com'，
        "avatar"：'xxxxxx'
    }
  }
返回示例--失败​
  {
    "status": 10001,
    "info":"error"，
    "message":"账号不存在，请输入正确账号"
  }，
  {
  "status":10002,
   "info":"error",
   "message":"密码错误，请重试"
  }

获取用户详情信息​

简要描述​
  • 获取用户详情信息
请求URL​
  • https://xxxxxxxxx/qiniuyun/user/detail?userId=001
请求方式​
  • get
返回示例--成功​
  {
    "status": 10000,
    "info": "success"，
    "data":[
        {
            userId:001,
            userName: '姓名',     //用户名
            avatar:'http:// ',   //头像
            collect: [
                 {
                   id:1,
                   title:'名字'，
                   summary:'这是一段简介',
                   url:'http://',
                   type:'热门',
                   userName:'发布者',
                   userId:002,
                   collectNum:99,
                   publishTime: （时间戳）,
                   duration: 360   (秒)
                }，
                {
                  id:2,
                  title:'名字'，
                  summary:'这是一段简介',
                  url:'http://',
                  type:'热门',
                  userName:'发布者',
                  userId:002,
                  collectNum:99,
                  publishTime: （时间戳）,
                  duration: 360   (秒)
                }
            ],//收藏列表
            follersNum:'10'， //关注人数
            follers:[
                {
                   userId:001,
                   userName: '姓名',
                   avatar:'http:// '
                }，
                {
                  userId:002,
                  userName: '姓名',
                  avatar:'http:// '
                 }
            ]
        }
    ]
  }
返回示例--失败​
  {
    "status": 10001,
    "info":"error"，
    "message":""
  }
返回参数说明​
参数名类型说明
userIdint用户id
usernamestring用户名
avatarstring头像
follersuser[]关注列表详情
follersNumint关注用户人数
collectvideo[]收藏视频详情
collectNumint收藏视频数

视频模块​

获取推荐视频​

简要描述​
  • 首页的随机推荐视频
请求URL​
  • https://xxxxxxxxx/qiniuyun/video/recommend?id=1
请求方式​
  • Get
请求参数说明​

id -> 视频分组序号，每次返回十条

返回示例--成功​
  {
    "status": 10000,
    "info": "success"，
    "message":"获取成功成功"
     "data"：[
        {
            id:1,
            title:'名字'，
            summary:'这是一段简介',
            url:'http://',
            type:'热门',
            userName:'发布者',
            userId:002,
            collectNum:99,
            publishTime: （时间戳）,
            duration: 360   (秒)
        }，
        {
            id:2,
            title:'名字'，
            summary:'这是一段简介',
            url:'http://',
            type:'热门',
            userName:'发布者',
            userId:002,
            collectNum:99,
            publishTime: （时间戳）,
            duration: 360   (秒)
        }
    ]
  }
返回示例--失败​
  {
    "status": 10001,
    "info":"error"，
    "message":"出错了"
  }
返回参数说明​
参数名类型说明
idint视频id
titlestring视频标题
summarystring视频简介
urlstring视频地址
typestring视频类型
userIdint用户id
userNamestring用户名
collectNumint收藏数
publishTimeDateTime发布时间
durationint视频时长 (秒钟)
备注:​

每次随机返回10条视频

获取视频评论​

简要描述​
  • 获取视频评论，用于
请求URL​
  • https://xxxxxxxxx/qiniuyun/video/comment?videoId=001
请求方式​
  • Get
请求参数说明​

videoId -> 视频id

返回示例--成功​
  {
    "status": 10000,
    "info": "success",
    "message":"获取成功成功"
     "data"：[
        {
            commentId:001,
            content:'这个男主角真帅',
            commentTime:1231232132,
            userName:'xsk',
            userId:001
        },
        {
            commentId:002,
            content:'这个女主角也好漂亮',
            commentTime:12314322132,
            userName:'xsk',
            userId:001
        }
    ]
  }
返回示例--失败​
  {
    "status": 10001,
    "info":"error"，
    "message":"出错了"
  }
返回参数说明​
参数名类型说明
commentIdint评论id
contentstring评论内容
commentTimeint评论时间，时间戳
userNamestring用户名
userIdstring用户id
